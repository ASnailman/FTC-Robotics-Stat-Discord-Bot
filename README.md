Discord bot that will show the stats of the team name specified.

Includes:
  - Team Name
  - Team Number
  - Current Global/State Rank
  - Non-Penalty OPR
  - Offensive Power Ranking (OPR)
  - Avg Score
  - Best Score

Use command: /teamstats [team name] [region] [year]

team name is REQUIRED or command won't work
region can be either any state or global, defaults to global, NOT REQUIRED
year can be set to any year, defaults to 2026, NOT REQUIRED
